§7  Unique Features — Deep Dive 
 
These four features do not appear anywhere in the case study brief. They are original product ideas. 
 
1. Safety Pulse ⭐ 
A single at-a-glance status line at the top of every page. Answers "how am I doing?" not "what are the alerts?" 
 
State Display 
0 active alerts "✓ You're caught up" — soft green, calm language 
N active, none affect me "N things need your attention" — soft amber 
N active, X affect me "N things need your attention — X directly affect you" — amber with sub-line 
 
2. Similar Alerts ⭐ 
After paste + AI categorize, shows 1-2 alerts from the DB corpus that match by category + keyword overlap. 
Reduces "is this real?" anxiety by linking new reports to known historical patterns. 
 
// src/lib/similarity.ts export function findSimilarAlerts(   c ategory: AlertCategory,   pastedText: 
string,   alerts: Alert[],   limit = 2 ): Alert[] {   const keywords = pastedText.toLowerCase()     
.split(/\W+/).filter(w => w.length > 4)     .filter(w => !STOPWORDS.includes(w))    return alerts     
.filter(a => a.category === category && a.id !== "new")     .map(a => ({       alert: a,       score: 
keywords.filter(k =>         (a.title + " " + a.description).toLowerCase().includes(k)       ).length     
}))     .filter(x => x.score > 0)     .sort((a, b) => b.score - a.score)     .slice(0, limit)     
.map(x => x.alert) } 
 
3. Threat DNA Fingerprint ⭐ 
A deterministic SVG generated from alert attributes. Same alert always produces the same fingerprint. Makes the 
list visually scannable — threats are recognisable at a glance. 
 
Attribute Visual Mapping 
category Base shape: Phishing=circle, Scam=diamond, Imposter=hexagon, Data 
breach=square, CVE=triangle, Local safety=shield, Other=dot 
severity Colour intensity: low=light, medium=mid, high=saturated, critical=dark+bold border 
source Pattern overlay: CISA=horizontal lines, PhishTank=diagonal lines, NVD=grid, 
User=dots 
title hash Slight rotation (0-15°) + size variation (±2px) — makes each alert unique 
 
// src/lib/fingerprint.ts export function generateFingerprint(alert: Alert): string {   const hash = 
simpleHash(alert.title)    // deterministic 0-100   const shape = SHAPES[alert.category]   const 
color = COLORS[alert.severity]   const pattern = PATTERNS[alert.source]   const rota tion = hash % 15   
const size = 20 + (hash % 4)    return `<svg viewBox="0 0 ${size} ${size}" ...>     
<defs>${pattern}</defs>     <${shape} fill="${color}" transform="rotate(${rotation})" .../>     
<${shape} fill="url(#pattern)" opacity="0.4" .../>   </sv g>` }  function simpleHash(str: string): 
number {   return str.split("").reduce((h, c) =>     ((h << 5) - h + c.charCodeAt(0)) & 0xff, 0   ) }
